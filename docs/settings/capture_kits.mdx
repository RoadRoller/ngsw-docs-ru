# Реагенты capture kit

Реагент capture kit - это таргетная панель, используемая при направленном отборе.
На странице "Реагенты Capture Kit" находятся все встроенные и загруженные в систему наборы
capture kit. Эти наборы используются в процессе анализа образцов, полученных в результате таргетного
секвенирования. Для анализируемого образца либо можно выбрать соответствующий ему реагент capture kit
на этапе загрузки образца, либо набор, наиболее подходящий образцу, будет выбран автоматически из всех наборов,
находящихся в системе. Кроме того, набор capture kit используется для
[фильтрации выявленных в образце SNVs/Indels](/settings/settings/variant_discovery_settings#фильтрация-по-capture-kit-и-её-отключение).

Чтобы открыть раздел с реагентами capture kit, перейдите на страницу "**Реагенты Capture Kit**" из блока
страниц "Настройки" через панель навигации слева:

<p align="center">
<img src={require('/img/capture_kits/ck_page.png').default} width="900"/>
</p>

На странице информация представлена в виде таблицы со следующими колонками:
- «ИМЯ» - содержит название реагента capture kit (имя загруженного файла capture kit);
- «ТИП» - содержит информацию о типе реагента: **Встроенный** (стандартные панели, часто применяемые
при секвенировании и встроенные в систему) или **Пользовательский** (загруженный пользователем);
- «ДЛИНА» - содержит информацию о суммарной длине реагента по всем геномным интервалам.

### Загрузить набор capture kit

1. Нажмите на <img src={require('/img/rus/upload_ck_btn.png').default} width="170"/>, чтобы открыть окно
загрузки файла с набором capture kit:

<p align="center">
<img src={require('/img/capture_kits/ck_upload_form.png').default} height="350"/>
</p>

2. В открывшемся окне выберите сборку референсного генома, соответствующую геномным интервалам
загружаемого набора capture kit: hg19 или hg38.
3. Загрузите файл capture kit в формате **BED** - файл с разделителями табуляцией или пробелами с тремя
обязательными столбцами: хромосома, начальная позиция интервала на хромосоме, конечная позиция интервала
на хромосоме. Для загрузки перетащите файл в окно или нажмите на него, чтобы открыть диспетчер файлов:

<p align="center">
<img src={require('/img/capture_kits/uploading_window.png').default} width="400"/>
</p>

4. Нажмите на <img src={require('/img/capture_kits/upload_btn.png').default} width="170"/>.
Загруженный capture kit отобразится в таблице и будет иметь тип "Пользовательский".

### Скачать набор capture kit

Чтобы скачать набор capture kit в формате BED, нажмите на его название в колонке "Имя".
Чтобы скачать загруженный файл capture kit в формате BED (файл до процессирования системой, в т.ч. до
перевода интервалов в геноме hg19 в интервалы в геноме hg38), наведите курсор на его строку и
нажмите на <img src={require('/img/rus/download_kit.png').default} width="20"/> (такая опция доступна
только для наборов, загруженных пользователем).

### Удалить набор capture kit

Удалить можно только загруженный пользователем набор, для встроенных наборов такой опции нет.
Чтобы удалить набор capture kit, наведите курсор на его строку и
нажмите на <img src={require('/img/rus/delete_btn.png').default} width="20"/>.
