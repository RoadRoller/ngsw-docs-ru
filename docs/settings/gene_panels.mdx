# Панели генов

В данном разделе можно добавлять панели генов, чтобы затем фильтровать по ним выявленные варианты.
Панель генов - это набор интересующих вас генов (например, генов, ассоциированных с определенным фенотипом).

Чтобы открыть раздел с генными панелями, перейдите на страницу "**Панели генов**" из блока
страниц "Настройки" через панель навигации слева:

<p align="center">
<img src={require('/img/gene_panels/gene_panels_page.png').default} width="900"/>
</p>

## Добавление панели генов

Доступны 4 способа добавления панели в систему:
1. [Выбор интересующих генов из всех генов (по названию или коду гена)](/settings/gene_panels#1-добавление-генов-в-панель-из-списка-всех-генов);
2. [Добавление генов, ассоциированных с определенным фенотипом из HPO](/settings/gene_panels#2-добавление-генов-в-панель-из-hpo);
3. [Импорт заранее подготовленного списка генов](/settings/gene_panels#3-импорт-списка-генов);
4. [Импорт панели из библиотеки самых распространенных клинических панелей](/settings/gene_panels#4-импорт-панели-из-библиотеки).

:::tip Совет
Во время добавления панели первый и второй способы можно комбинировать.
:::

### 1. Добавление генов в панель из списка всех генов

1. Нажмите на <img src={require('/img/rus/add_gene_panel.png').default} width="170"/>, чтобы открыть форму
добавления панели генов:

<p align="center">
<img src={require('/img/gene_panels/add_gene_panel_form.png').default} height="700"/>
</p>

2. В открывшейся форме введите название панели генов в поле **Имя**.
Можете также добавить описание панели.

3. Добавьте гены в панель. Для этого введите в окно поиска официальное название гена или код гена из Ensembl (ENSG...).
Среди найденных генов выберите один или несколько нужных, нажмите на его/их строку/и, а затем
на кнопку <img src={require('/img/gene_panels/add_btn.png').default} width="40"/>:

<p align="center">
<img src={require('/img/gene_panels/add_gene.png').default} height="350"/>
</p>

Среди генов, добавленных в панель, можно осуществлять поиск, а также убирать гены из панели -
для этого нажмите на строку гена, а затем
на кнопку <img src={require('/img/gene_panels/remove_gene.png').default} width="40"/>:

<p align="center">
<img src={require('/img/gene_panels/remove_example.png').default} height="350"/>
</p>

4. После того, как вы добавили все нужные гены в панель,
нажмите на <img src={require('/img/rus/create_btn.png').default} width="130"/>.
Панель генов будет добавлена в систему.

### 2. Добавление генов в панель из HPO

1. Нажмите на <img src={require('/img/rus/add_gene_panel.png').default} width="170"/>, чтобы открыть форму
добавления панели генов:

<p align="center">
<img src={require('/img/gene_panels/add_gene_panel_form.png').default} height="700"/>
</p>

2. В открывшейся форме введите название панели генов в поле **Имя**.
Можете также добавить описание панели.

3. Нажмите на <img src={require('/img/gene_panels/add_from_hpo.png').default} width="280"/>.
Откроется окно поиска термина из словаря фенотипических аномалий
человека <a href="https://www.ebi.ac.uk/ols/ontologies/hp">Human Phenotype Ontology (HPO)</a>:

<p align="center">
<img src={require('/img/vv/hpo_search.png').default} width="800"/>
</p>

В выпадающем списке представлены основные группы фенотипов.
Вы можете добавить в панель как гены, ассоциированные с целой группой фенотипов, так и гены,
ассоциированные с конкретным фенотипом. Фенотип можно найти в списке терминов определенной группы фенотипов
или через поиск по названию или идентификатору из HPO.
Например, при поиске заболевания “glioma” находится несколько терминов:

<p align="center">
<img src={require('/img/vv/glioma_search.png').default} width="800"/>
</p>

Выбрав один из них (например, "Glioma"), вы окажетесь на странице этого заболевания:

<p align="center">
<img src={require('/img/vv/glioma_page.png').default} width="800"/>
</p>

Слева даны определение заболевания и комментарий. В середине представлена иерархия заболевания:
выше выбранного заболевания Glioma находятся его родительские фенотипы
(те группы фенотипов, которые включают Glioma), а ниже - его дочерние
фенотипы (заболевания, которые входят в группу Glioma).
Справа представлены гены, ассоциированные с данным заболеванием (в данном случае было найдено 45 генов).

Для того чтобы выйти из формы без добавления генов в панель,
нажмите на <img src={require('/img/gene_panels/cancel_btn.png').default} width="150"/>.
Если вы добавили гены из HPO в панель, они будут отображены в списке справа. Среди них можно осуществлять поиск,
а также убирать ненужные гены из панели - для этого нажмите на строку гена, а затем
на кнопку <img src={require('/img/gene_panels/remove_gene.png').default} width="40"/>:

<p align="center">
<img src={require('/img/gene_panels/remove_hpo_gene.png').default} height="350"/>
</p>

4. После того, как вы добавили все нужные гены в панель,
нажмите на <img src={require('/img/rus/create_btn.png').default} width="130"/>.
Панель генов будет добавлена в систему.

### 3. Импорт списка генов

1. Нажмите на <img src={require('/img/rus/import_gene_list.png').default} width="170"/>, чтобы открыть форму
импорта списка генов:

<p align="center">
<img src={require('/img/gene_panels/import_form.png').default} height="500"/>
</p>

2. В открывшейся форме введите название панели генов в поле **Имя**.
Можете также добавить описание панели.

3. Введите список генов в окно **Гены**. Список должен включать только официальные названия генов и/или их
коды из Ensembl (ENSG...), разделенные запятыми или новой строкой. Предпочтительнее использовать коды из Ensembl либо
уникальные синонимы генов, соответствующие только одному коду Ensembl,
так как при использовании названий генов может возникнуть неоднозначность из-за существования
генов альтернативной последовательности ДНК человека (в приведенном примере у гена *MUC6* их три:
[ENSG00000277281](https://www.ensembl.org/Homo_sapiens/Gene/Summary?db=core;g=ENSG00000277281),
[ENSG00000277518](https://www.ensembl.org/Homo_sapiens/Gene/Summary?db=core;g=ENSG00000277518),
[ENSG00000283350](https://www.ensembl.org/Homo_sapiens/Gene/Summary?db=core;g=ENSG00000283350)):

<p align="center">
<img src={require('/img/gene_panels/ambigous_gene.png').default} width="600"/>
</p>

4. Нажмите на <img src={require('/img/rus/create_btn.png').default} width="130"/> для добавления
панели генов в систему.

### 4. Импорт панели из библиотеки

1. Нажмите на <img src={require('/img/rus/import_from_library.png').default} width="170"/>, чтобы открыть форму
импорта панели:

<p align="center">
<img src={require('/img/gene_panels/import_from_library.png').default} height="700"/>
</p>

2. В открывшейся форме выберите нужные панели, отметив их галочками:

<p align="center">
<img src={require('/img/gene_panels/checked_panels.png').default} width="600"/>
</p>

3. Нажмите на <img src={require('/img/gene_panels/import_btn.png').default} width="170"/>.

## Редактирование панели генов

Чтобы добавить или убрать гены из добавленной панели, нажмите на её строку на странице "Панели генов".
В открывшейся форме панели можно поменять название и описание панели,
убрать и добавить гены в панель (добавить гены можно либо из списка всех генов, либо из HPO).
После внесения изменений нажмите на <img src={require('/img/group_analysis/save_btn.png').default} width="150"/>.

## Удаление панели генов

Чтобы удалить панель генов из системы, наведите курсор на её строку на странице "Панели генов" и нажмите на
появившуюся кнопку <img src={require('/img/rus/delete_btn.png').default} width="21"/>.
