# Основные шаблоны отчетов

В данном разделе описаны основные шаблоны отчетов, которые используются для представления результатов анализа.
Некоторые из них могли быть добавлены в ваш профиль администратором при регистрации в Системе. Если вам понадобятся
другие шаблоны отчетов, описанные здесь, вы можете обратиться к администратору или добавить их самостоятельно,
как описано ниже.

## 1. Отчет для анализа глиомы

- клинический отчет о характеристиках глиом, основанный на онко-маркерах, выявленных в образце, и базе знаний,
созданной на основе эскпертных знаний и научных публикаций по данной тематике;
- подходит для представления результатов анализа образцов глиомы в любом формате файлов.

### Условия построения отчета

Отчет для анализа глиомы строится для образца, если выполняются следующие условия:
1. Образец загружен как образец опухолевой ткани.
2. Анализ образца успешно завершился (то есть все стадии анализа имеют статус "Завершено").
3. Для одиночного образца опухоли (tumor-only) успешно завершилась хотя бы одна из следующих стадий: "Аннотация
соматических SNV/Indels", "Аннотация герминальных SNV/Indels" или "Выявление вариации числа копий". Если же у вас
набор образцов "опухолевый + соответствующий ему неопухолевый образец" (tumor-normal), то для построения отчета
необходимо, чтобы для опухолевого образца успешно завершилась стадия "Аннотация соматических
SNV/Indels", а для неопухолевого - "Аннотация герминальных SNV/Indels".
3. Шаблон отчета активен.
4. Шаблон отчета был добавлен раньше, чем был обработан образец.

### Добавление шаблона отчета для анализа глиомы

Чтобы добавить шаблон отчета для анализа глиомы в cистему, вы можете либо импортировать его,
либо создать самостоятельно.

#### Импорт шаблона отчета для анализа глиомы

Скачайте файл шаблона (<a target="_blank" href="/reports/Отчет для анализа глиомы.json" download="Отчет для анализа глиомы.json">Отчет для анализа глиомы</a>)

## 2. Отчет по CNV

- отчет по результатам анализа числа копий хромосом в исследованном образце или образцах пациента;
- влияет на построение [отчета по CNV для образца](/results/cnv-report/sample-cnv-report) и
[отчета по CNV для пациента (для всех образцов пациента)](/results/cnv-report/patient-cnv-report);
- подходит для представления результатов анализа неопухолевых образцов в формате FASTQ или BAM.

### Условия построения отчета

Отчет по CNV строится для образца, если выполняются следующие условия:
1. Образец загружен как образец неопухолевой ткани.
2. Анализ образца успешно завершился (то есть все стадии анализа имеют статус "Завершено"). Если анализ завершился
с ошибкой, то в "Отчете по CNV" для пациента будет соответствующее сообщение для образца, а "Отчет по CNV" для образца
не построится.
3. Для образца успешно завершилась стадия "Выявление вариации числа копий". Если стадия завершилась
с ошибкой, то в "Отчете по CNV" для пациента будет соответствующее сообщение для образца, а "Отчет по CNV" для образца
не построится. Если стадия исключена из анализа, то отчет для образца не построится, а в отчете для пациента образец
упомянут не будет. Если стадия выключена у всех образцов пациента, то "Отчет по CNV" для пациента не построится.
3. Шаблон отчета активен.
4. Шаблон отчета был добавлен раньше, чем был обработан образец.
