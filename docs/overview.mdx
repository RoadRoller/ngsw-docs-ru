## Образцы {#samples}

Начальная страница "Образцы" представляет все образцы, которые загружены в пациентов
или запуски, которые находятся не в архиве. Если для образца определен как пациент, так и запуск,
то он будет отображаться на странице "Образцы", если хотя бы одна из этих сущностей не архивирована.

#### На странице "Образцы" информация представлена в виде таблицы со следующими колонками: {#sample-columns}
- «ОБРАЗЕЦ» - информация о загруженных в систему образцах, которые принадлежат пациентам или

обработанный образец
неопухолевой ткани;<br></br><img src={require('/img/rus/normal_problems.png').default} width="18"/> - образец
неопухолевой ткани, обработанный с
некритичными проблемами;<br></br><img src={require('/img/rus/normal_failed.png').default} width="18"/> - образец
неопухолевой ткани, обработанный с ошибкой.
2. Имя файла образца без расширения и постфиксов (если файлы названы
согласно <a href="https://support.illumina.com/help/BaseSpace_Sequence_Hub_OLH_009008_2/Source/Informatics/BS/NamingConvention_FASTQ-files-swBS.htm">конвенции Illumina</a>).
В случае парного секвенирования приводится только имя первичного файла.
Нажав на имя файла, можно открыть <a href="/results/results">страницу образца</a>.
3. Формат файла: <i>FASTQ</i> или <i>VCF</i>. Для образцов, загруженных в формате BAM, или данных
генотипирования, загруженных в формате TXT, TSV, будет приведен тот формат, в который были переведены
исходные данные для дальнейшего анализа, т.е. FASTQ и BAM, соответственно.
4. Тип секвенирования: <i>Панель</i> (секвенирование с помощью таргетной панели), <i>WES</i> (whole-exome sequencing;
полноэкзомное секвенирование), <i>WGS</i> (whole-genome sequencing; полногеномное секвенирование), <i>Low-pass WGS</i>
(полногеномное секвенирование с низким покрытием), <i>Unknown</i> (тип секвенирования может быть не распознан для
некоторых образцов, загруженных в формате VCF). Определяется автоматически, может быть основан на типе секвенирования
и реагентах Capture Kit, выбранных на этапе создания набора образцов. Отображается только после определения, т.е. после
успешного завершения стадии "Определение набора capture kit" для FASTQ файлов и стадий "Идентификация"
или "Аннотация SNVs/Indels" для VCF файлов.
5. Размер образца. В случае парного секвенирования будет отображен суммарный размер файлов образца.
- «ПАЦИЕНТ / ЗАПУСК» - код пациента (обозначен <img src={require('/img/rus/patient_icon.png').default} width="20"/>)
и/или название запуска (обозначен <img src={require('/img/rus/run_icon.png').default} width="20"/>),
которым принадлежит образец. Нажав на код пациента или название запуска, можно
открыть, соответственно, <a href="/add-patient#patient-page">страницу пациента</a>
или <a href="/runs#run-page">страницу запуска</a>.
- «РЕЗУЛЬТАТЫ» - выводы интерпретации образца: информация о количестве обнаруженных в образце
и <a href="/results/main/snvs-indels/variant-viewer#include-to-report">добавленных в отчёт</a> вариантов
определенного класса патогенности (патогенных: <img src={require('/img/results/pathogenic.png').default} width="49"/>,
вероятно патогенных: <img src={require('/img/results/likely_pathogenic.png').default} width="50"/>,
неопределенной значимости: <img src={require('/img/results/uncertain.png').default} width="50"/>,
вероятно доброкачественных: <img src={require('/img/results/likely_benign.png').default} width="50"/>,
доброкачественных: <img src={require('/img/results/benign.png').default} width="50"/>).
Результаты отображаются, если интерпретация образца была завершена (как описано ниже)
или если вариант с определенной патогенностью был добавлен в отчёт.
Если интерпретация была не завершена, но варианты с определенным классом патогенности были добавлены в отчёт, то
результатом будет "<i>Интерпретация не завершена</i>". Если интерпретация образца была завершена,
но никакие варианты с определенным классом патогенности не были добавлены в отчёт, то результатом
будет "<i>Патогенных вариантов не обнаружено</i>".
- «СОЗДАН» - дата начала загрузки данных образца в систему в формате д.М.гггг;
- «ИЗМЕНЕН» - дата последних изменений в данных образца в формате д.М.гггг.

При наведении курсора на строку образца появляются следующие активные элементы:
- <img src={require('/img/rus/sample_vv.png').default} width="17"/> - кнопка для перехода
в <a href="/results/main/snvs-indels/variant-viewer">SNV Viewer</a> образца. Для опухолевого образца
это SNV Viewer c соматическими SNVs/Indels, в котором также можно увидеть и SNVs/Indels из SNV Viewer
с герминальными мутациями, если поменять значение фильтра по типу мутации с Somatic на Germline или
сбросить фильтрацию по нему. Для неопухолевого образца это SNV Viewer только с герминальными SNVs/Indels,
выявленными в этом образце. Кнопка отображается, если для образца успешно завершилась стадия
"Аннотация SNVs/Indels".
- <img src={require('/img/rus/sample_cnvv_segments.png').default} width="22"/> - кнопка для перехода
в CNV Viewer образца с сегментами, выявленными в этом образце. Кнопка отображается, если для образца
успешно завершилась стадия "Выявление вариации числа копий".
- <img src={require('/img/rus/sample_cnvv_bins.png').default} width="22"/> - кнопка для перехода
в CNV Viewer образца с бинами, выявленными в этом образце. Кнопка отображается, если для образца
успешно завершилась стадия "Выявление вариации числа копий".
- <img src={require('/img/rus/sample_complete_interpretation.png').default} width="22"/> или <img src={require('/img/rus/sample_restore_interpretation.png').default} width="22"/> - кнопки
для завершения и восстановления интерпретации образца, соответственно. После нажатия на кнопку завершения
появится окно для подтверждения:

<p align="center">
<img src={require('/img/vv/interpretation_window.png').default} height="600"/>
</p>

После того, как вы нажмёте <img src={require('/img/custom-annot/ok_btn.png').default} width="130"/>, вывод
интерпретации из этого окна отобразится в колонке «РЕЗУЛЬТАТЫ».

- <img src={require('/img/rus/delete_btn.png').default} width="20"/> - кнопка для
удаления образца из системы. После нажатия на неё появится окно для подтверждения:

<p align="center">
<img src={require('/img/rus/delete_sample.png').default} width="450"/>
</p>

После подтверждения действий нажатием на <img src={require('/img/rus/delete.png').default} width="150"/>
образец удалится как со страницы "Образцы", так и со страницы пациента и/или запуска, которому он
принадлежал.

Вверху страницы находятся:
- Окно поиска, позволяющее искать образцы по имени файла и по коду
пациента или названию запуска, которому они принадлежат:
<p align="center"><img src={require('/img/rus/search_sample.png').default} width="500"/></p>
- Кнопка <a href="/add-patient">добавления пациента</a>
в систему <img src={require('/img/rus/add_patient_btn_from_samples.png').default} width="180"/>.
- Кнопка <a href="/runs">добавления запуска</a>
в систему <img src={require('/img/rus/add_run_btn.png').default} width="170"/>.

</details>

- "***Профиль***" - открывает страницу ["Настройки пользователя"](/settings/profile), где
находятся ваши персональные настройки аккаунта (email, логин, пароль), настройки панели
навигации Программы, страницы пациента, SNV Viewer и CNV Viewer.

- "***Сообщить о проблеме***" - открывает окно для отправки информации о возникшей у вас проблеме
администратору Системы:

<p align="center">
<img src={require('/img/rus/error_window.png').default} width="600"/>
</p>

- "***О программе***" - открывает окно с информацией о Программе, включая информацию о версиях сервера и
клиента и версиях баз данных для аннотации.
- "***Выйти***" - осуществляет выход пользователя из Системы.
