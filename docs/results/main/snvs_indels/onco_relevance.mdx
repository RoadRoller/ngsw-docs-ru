# Онкозначимость варианта

Онкозначимость варианта определяется тем, что он известен как участник в развитии онкологических заболеваний.
Для таких онкозначимых вариантов были собраны клинические рекомендации по лекарственному лечению
злокачественных опухолей и поддерживающей терапии при онкологии, разработанные группами экспертов, изучающих
раковые заболевания: [Российским обществом клинической онкологии (RUSSCO)](https://www.rosoncoweb.ru/),
[Ассоциацией онкологов России (АОР)](https://oncology-association.ru/),
[Министерством здравоохранения Российской Федерации (МЗ РФ)](https://cr.minzdrav.gov.ru/),
[National Comprehensive Cancer Network (NCCN)](https://www.nccn.org/),
[European Society for Medical Oncology (ESMO)](https://www.esmo.org/).

В Variant Viewer можно добавить **колонку "Онкозначимость"** (не входит в число колонок по умолчанию) в
[режиме редактирования колонок](/results/main/snvs_indels/edit_table_columns#добавление-колонок).
Если вариант онкозначимый, то значением этой колонки для
него будет кнопка <img src={require('/img/vv/onco_relevance_btn.png').default} width="30"/>.
Нажав на эту кнопку, вы попадете на вкладку "Клинические рекомендации" страницы детальной информации о варианте.

:::note На заметку
По значению колонки "Онкозначимость" можно отфильтровать варианты, воспользовавшись фильтром
["Onco Relevance"](/results/main/snvs_indels/advanced_filtering#2-фильтры-с-выбором-одного-значения)
в режиме расширенного поиска вариантов.
:::

### Уточнение рекомендаций для пациента

Если вы хотите специфицировать рекомендации для конкретного пациента, укажите его **онкологическое заболевание**,
**стадию** (I-IV) с **уточнением** (рецидив, прогрессирование, неоперабельная, метастазы в головной мозг) и
**линию терапии** (1-4 линия, адъювантная терапия, периоперационная, предоперационная или индукционная химиотерапия).
Это можно сделать на [странице образца](/results/main/main#2-информация-об-образце) или
на [вкладке "Образец"](/results/main/snvs_indels/snv_detaile_page#2-вкладка-образец) страницы детальной
информации о варианте.

К выбору доступны следующие **онкологические заболевания** (на данный момент в Genomenal только варианты,
ассоциированные с этими заболеваниями, отмечены как онкозначимые и только для них есть клинические рекомендации):
- НМРЛ - немелкоклеточный рак легкого;
- Меланома;
- Рак поджелудочной железы;
- РМЖ - рак молочной железы;
- Рак яичников;
- Рак простаты;
- Анапластический рак щитовидной железы;
- Колоректальный рак.

Рекомендации, соответствующие выбранным заболеванию, стадии, уточнению и линии терапии,
отмечены плашкой <img src={require('/img/vv/satisfies_parameters.png').default} width="250"/>:
