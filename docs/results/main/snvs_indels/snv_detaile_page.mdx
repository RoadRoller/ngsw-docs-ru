# Страница детальной информации о варианте

Страница детальной информации о варианте содержит:
- [Подробную аннотацию варианта](/results/main/snvs_indels/snv_detaile_page#1-вкладка-аннотация);
- [Информацию об образце и пациенте](/results/main/snvs_indels/snv_detaile_page#2-вкладка-образец);
- [ACMG классификацию](/results/main/snvs_indels/snv_detaile_page#3-вкладка-acmg-классификация);
- [Научные публикации, ассоциированные с вариантом](/results/main/snvs_indels/snv_detaile_page#4-вкладка-литература).

Чтобы перейти на страницу детальной информации о варианте из Variant Viewer,
нажмите на <img src={require('/img/vv/details_info_btn.png').default} width="25"/> в строке варианта.

На открывшейся странице сверху приведена основная информация о варианте:
- Название гена, в котором располагается вариант;
- Нуклеотидная замена - либо в [номенклатуре HGVS](https://varnomen.hgvs.org/) (HGVSc = префикс (“c.” (coding; для
замены в кодирующей последовательности) или “n.” (non-coding; для замены в некодирующей последовательности)) +
геномная позиция замещенного нуклеотида + референсный аллель > альтернативный аллель), либо в виде замены
(префикс "g." (genomic; для линейной геномной референсной последовательности) + начальная позиция варианта в геноме +
референсный аллель > альтернативный аллель);
- Координаты варианта в геноме (хромосома + начальная позиция);
- Нуклеотидная замена (референсный аллель > альтернативные аллели/аллель);
- Идентификатор транскрипта, выбранного главным для данного варианта, из базы
[Ensembl](https://www.ensembl.org/Homo_sapiens/Info/Index) (ENSTxxxxx);
- Номер экзона в транскрипте, который вариант затронул;
- Эффект варианта (последствия варианта для генов; см. подробно
[здесь](/results/main/snvs_indels/variant-viewer#эффекты-варианта-в-аннотации));
- Ссылка на вариант в базе <a href="https://www.ncbi.nlm.nih.gov/snp/">dbSNP</a>.

Чтобы ***добавить вариант в отчет*** со страницы детальной информации,
включите соответствующий переключатель <img src={require('/img/vv/to_report.png').default} width="180"/> -
вариант будет включен в отчет, и
откроется [вкладка "Отчеты"](/results/main/snvs_indels/snv_detaile_page#5-вкладка-отчеты).
Включенные в отчет варианты будут входить в блоки отчетов с типом
["Выбранные пользователем SNVs/Indels для добавления в отчет"](/settings/report-templates/report-templates#типы-блоков-шаблона-отчета-и-их-редактирование).
Для исключения варианта из отчета выключите переключатель обратно.
Также включенные в отчет варианты можно удалить из отчета на странице самого отчета.

Чтобы ***закрепить вариант*** в таблице Variant Viewer (подробно о закреплении можно
почитать [здесь](/results/main/snvs_indels/variant-viewer#закрепление-вариантов))
со страницы детальной информации, нажмите на <img src={require('/img/vv/pin_on_details_page.png').default} width="40"/>.
Для открепления нажмите ещё раз на ту же
кнопку <img src={require('/img/vv/unpin_on_details_page.png').default} width="40"/>.

Чтобы открыть встроенный ***модуль визуализации варианта в геноме (IGV)***,
нажмите на <img src={require('/img/vv/igv_btn.png').default} width="40"/>.

Чтобы ***закрыть страницу*** детальной информации о варианте,
нажмите на <img src={require('/img/vv/close_page.png').default} width="40"/>.

Шкала патогенности в правом верхнем углу страницы описана в
[соответствующем разделе](/results/main/snvs_indels/pathogenicity#определение-патогенности-на-вкладке-acmg-классификация).

текста элемента, например, одно предложение из целого резюме статьи).

## 1. Вкладка "Аннотация"

На вкладке "Аннотация" приведены те разделы аннотации, которые включают хотя бы одно не пустое поле для варианта.
Разделы аннотации соответствуют вкладкам панели детальной информации, подробное описание можно почитать в
[соответствующем разделе](/results/main/snvs_indels/snv_details_panel#вкладки-панели-детальной-информации).

## 2. Вкладка "Образец"

На вкладке образец находится два раздела:
1. **Информация об образце** - соответствует информации об образце
со [вкладки "Главное" страницы образца](/results/main/main#2-информация-об-образце). Данные можно добавить или
поменять как на странице образца, так и здесь, на вкладке "Образец" страницы детальной информации о варианте.
Раздел содержит следующую информацию об образце:
- ТИП СЕКВЕНИРОВАНИЯ: Панель (секвенирование с помощью таргетной панели), WES (whole-exome sequencing; полноэкзомное
секвенирование), WGS (whole-genome sequencing; полногеномное секвенирование), Low-pass WGS (полногеномное
секвенирование с низким покрытием), UNKNOWN (тип секвенирования может быть не распознан для некоторых образцов,
загруженных в формате VCF). Определяется автоматически, может быть основан на типе секвенирования и
реагентах Capture Kit, выбранных на этапе создания набора образцов. Тип можно поменять
на [вкладке "Параметры"](/results/parameters) ([параметр "Sequencing Type"](/settings/settings/alignment_settings#определение-типа-секвенирования):
опция "WGS" сооветствует типу "WGS" или "Low-pass WGS", опция "Направленный отбор" соответствует либо типу
"Панель", либо "WES").
- ДАТА СБОРА ОБРАЗЦОВ - дата в формате М/д/гггг, можно указать, нажав на поле значения.
- РЕАГЕНТЫ CAPTURE KIT - это таргетная панель, используемая при направленном отборе. Определяется автоматически
Genomenal, либо самим пользователем на этапе загрузки образца.
Если capture kit определен неправильно, его можно поменять на вкладке "Параметры".
При анализе можно использовать как встроенные в систему реагенты capture kit (стандартные панели, часто применяемые
при секвенировании), так и загруженные пользователем (на странице "Реагенты Capture Kit" в блоке страниц "Настройки").
- ЛОКАЛИЗАЦИЯ ОБРАЗЦА (ОРГАН) - часть организма, из которой был взят образец для секвенирования. Можно указать,
нажав на поле значения.
- МЕТОДИКА - методика выделения образца, можно указать, нажав на поле значения.
- ОНКОЛОГИЧЕСКОЕ ЗАБОЛЕВАНИЕ, СТАДИЯ и УТОЧНЕНИЕ - онкологическое заболевание, его стадия и
уточнение стадии, которые были определены у пациента на момент взятия материала образца секвенирования.
- ЛИНИЯ ТЕРАПИИ - линия терапии, используемая для лечения заболевания пациента на момент взятия материала образца
секвенирования.
- ВЫВОДЫ ИНТЕРПРЕТАЦИИ - содержит информацию о количестве обнаруженных в образце и добавленных в отчет вариантов
определенного класса патогенности (патогенных, вероятно патогенных, неопределенной значимости, доброкачественных,
вероятно доброкачественных). После добавления вариантов с определенным классом патогенности в отчет
вывод интерпретации будет "Интерпретация не завершена". Если интерпретация образца была завершена,
но никакие варианты с определенным классом патогенности не были добавлены в отчет, то вывод интерпретации
будет следующим: "Патогенных вариантов не обнаружено". Подробнее об интерпретации образца читайте ниже.
2. **Информация о пациенте** - соответствует информации о пациенте
со [страницы пациента](/add_patient#информация-о-пациенте). Данные можно добавить или
поменять как на странице пациента, так и здесь, на вкладке "Образец" страницы детальной информации о варианте.
Раздел содержит следующую информацию об образце:
- КОД ПАЦИЕНТА - код, введенный при создании пациента (можно отредактировать, нажав на код). Может включать
любые символы.
- ДАТА РЕГИСТРАЦИИ - дата создания пациента в формате М/д/гггг.
- ИМЯ, ФАМИЛИЯ пациента. Можно указать, нажав на поля значений.
- ДАТА РОЖДЕНИЯ пациента в формате М/д/гггг, можно указать, нажав на поле значения.
- ПОЛ пациента: мужчина, женщина. Пол можно указать, нажав на поле значение и выбрав нужное значение
 из выпадающего списка.
- ПОСТАВЛЕННЫЙ ДИАГНОЗ - диагноз пациента, поставленный на момент взятия материала образца секвенирования.
Можно указать, нажав на поля значений.

### Интерпретация образца: завершение и возобновление

Интерпретация образца - интерпретация патогенности вариантов образца. Для учета варианта, выявленного
в образце, в выводах интерпретации, нужно:
- Определить для него класс патогенности (читайте
подробно в [соответствующем разделе](/results/main/snvs_indels/pathogenicity));
- Добавить его в отчет (со [страницы Variant Viewer](/results/main/snvs_indels/variant-viewer#включение-вариантов-в-отчет)
или со страницы детальной информации о варианте - см. выше).

:::note На заметку
Варианты можно будет добавить в интерпретацию и уже после её завершения.
:::

Чтобы ***завершить интерпретацию***,
нажмите на <img src={require('/img/vv/finish_interpretation.png').default} width="210"/> на вкладке "Образец"
страницы детальной информации о варианте. В появившемся окне будет приведено количество
обнаруженных в образце и добавленных в отчет вариантов определенного класса патогенности
(патогенных: <img src={require('/img/results/pathogenic.png').default} width="48"/>,
вероятно патогенных: <img src={require('/img/results/likely_pathogenic.png').default} width="48"/>,
неопределенной значимости: <img src={require('/img/results/uncertain.png').default} width="48"/>,
доброкачественных: <img src={require('/img/results/benign.png').default} width="48"/>,
вероятно доброкачественных: <img src={require('/img/results/likely_benign.png').default} width="48"/>).
Если никакие варианты с определенным классом патогенности не были добавлены в отчет,
то вывод интерпретации будет следующим: "Патогенных вариантов не обнаружено".

<p align="center">
<img src={require('/img/vv/interpretation_window.png').default} height="500"/>
</p>

В этом же окне можно указать онкологическое заболевание, его стадию и уточнение стадии, которые были определены
у пациента на момент взятия материала образца секвенирования, а также линию терапии, используемую для лечения
заболевания пациента на момент взятия материала образца секвенирования.

Для завершения интерпретации нажмите <img src={require('/img/custom-annot/ok_btn.png').default} width="130"/>.

Для ***возобновления интерпретации***
нажмите на <img src={require('/img/vv/restart_interpretation.png').default} width="210"/> на вкладке "Образец"
страницы детальной информации о варианте.

нажмите на <img src={require('/img/vv/search_in_google_scholar.png').default} width="230"/>.

## 5. Вкладка "Отчеты"

На вкладке можно включить вариант в отчет, посмотреть, как информация о нём будет выглядеть в отчете,
и экспортировать отчет с этим вариантом.

Чтобы ***добавить вариант в отчет*** со страницы детальной информации,
включите соответствующий переключатель <img src={require('/img/vv/to_report_2.png').default} width="180"/>.
Включенные в отчет варианты будут входить в блоки отчетов с типом
["Выбранные пользователем SNVs/Indels для добавления в отчет"](/settings/report-templates/report-templates#типы-блоков-шаблона-отчета-и-их-редактирование).
Для исключения варианта из отчета выключите переключатель обратно.
Также включенные в отчет варианты можно удалить из отчета на странице самого отчета.

После добавления варианта в отчет станет доступен предварительный просмотр того, как вариант будет
выглядеть в отчете:

<p align="center">
<img src={require('/img/vv/variant_review.png').default} width="900"/>
</p>

Варианты в отчете занесены в таблицу со следующими колонками:
1. **Ген** - название гена, в котором располагается вариант. При нажатии на ген появится окно со всеми
транскриптами, известными для гена:

<p align="center">
<img src={require('/img/vv/transcripts.png').default} width="600"/>
</p>

Описание колонок таблицы с транскриптами можно найти в описании
[раздела "Транскрипты" на панели детальной информации](/results/main/snvs_indels/snv_details_panel#common---основная-информация-о-варианте).

2. **Позиция** - координаты варианта в геноме (хромосома + начальная позиция).
3. **Генетический вариант** - нуклеотидная и аминокислотная замены
в [номенклатуре HGVS](https://varnomen.hgvs.org/). Нуклеотидная: префикс “c.” (coding; для замены в
кодирующей последовательности) или “n.” (non-coding; для замены в некодирующей последовательности) + геномная
позиция замещенного нуклеотида + референсный аллель > альтернативный аллель. Аминокислотная: префикс “p.” (protein) +
референсная аминокислота + позиция аминокислоты в белке + новая аминокислота, получившаяся в результате замены.
4. **Транскрипт** - идентификатор основного транскрипта гена из базы [RefSeq](https://www.ncbi.nlm.nih.gov/nuccore/)
(NM_xxxxxx.x). При нажатии на ген появится такое же окно со всеми транскриптами, известными для гена, как и
при нажатии на поле "Ген".
5. **Эффект варианта** - последствия варианта для генов. Подробное описание возможных значений
можно посмотреть [здесь](/results/main/snvs_indels/variant-viewer#эффекты-варианта-в-аннотации).
6. **Содержание аллеля** - частота альтернативного аллеля для данного образца (в процентах).
7. **Глубина прочтения** - общее количество прочтений последовательности, перекрывающих позицию варианта, для
данного образца.
8. **Внешние ссылки** - ссылки на вариант в базах данных [dbSNP](https://www.ncbi.nlm.nih.gov/snp/),
[ClinVar](https://preview.ncbi.nlm.nih.gov/clinvar/) и [COSMIC](https://cancer.sanger.ac.uk/cosmic/) (если база
была [загружена в виде пользовательской аннотации](/settings/custom-annotations#2-определение-типа-пользовательской-аннотации)).
