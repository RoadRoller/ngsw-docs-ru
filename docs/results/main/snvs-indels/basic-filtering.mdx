## Основные параметры фильтрации {#basic-filtering-columns}

:::caution На заметку
На [странице "Настройки пользователя"](/settings/profile#basic-mode-filters)
можно поменять выбор основных параметров фильтрации. Далее подробно рассказано о тех параметрах,
которые добавлены в настройках по умолчанию.
:::

### 1. Тип мутации {#origin}

- Фильтр вариантов по типу мутации: соматическая (Somatic) или герминальная (Germline).

:::note На заметку
Параметр фильтрации "Тип мутации" не отображается в базовом режиме поиска в
[SNV Viewer с герминальными мутациями](/results/main/snvs-indels/variant-viewer).
:::

Как здесь, так и во всех последующих случаях, <b><i>алгоритм фильтрации</i></b> один:
- Нажмите на окно фильтрации;
- Из появившегося списка выберите нужную опцию фильтрации либо введите интересующую вас опцию в окно
поиска и выберите из найденных.

Готово! Фильтр применен. То, какой выбран фильтр, можно увидеть в окошке (в данном случае выбран тип мутации
"Somatic"):

<p align="center">
<img src={require('/img/vv/set_option.png').default} width="150"/>
</p>

### 2. Панель генов {#gene-panel}

- Фильтр вариантов по локализации в определенных генах из панели.
Вы можете добавить панель генов <a href="/settings/gene-panels">на странице "Панели генов"</a>
или импортировать панель из библиотеки самых распространенных клинических панелей. Для импорта
нажмите на "<i>Импорт из библиотеки</i>":

<p align="center">
<img src={require('/img/vv/gene_panel.png').default} width="300"/>
</p>

В открывшейся форме выберите нужные панели, отметив их галочками:

<p align="center">
<img src={require('/img/gene_panels/checked_panels.png').default} width="600"/>
</p>

и нажмите на <img src={require('/img/gene_panels/import_btn.png').default} width="150"/>, чтобы добавить панель.

### 3. Ген {#gene}

- Фильтр вариантов по локализации в транскрипте гена. Транскрипт приведен в скобках рядом с официальным названием гена.
Можно отфильтровать варианты сразу по нескольким генам. Для этого отметьте галочками нужные гены:

<p align="center">
<img src={require('/img/vv/filter_by_gene.png').default} width="200"/>
</p>

Если для гена известно несколько транскриптов, в списке представлены все:

<p align="center">
<img src={require('/img/vv/many_transcripts.png').default} height="400"/>
</p>

### 4. HPO {#hpo}

- Фильтр вариантов по локализации в генах, ассоциированных с терминами из словаря фенотипических аномалий
человека <a href="https://www.ebi.ac.uk/ols/ontologies/hp">Human Phenotype Ontology (HPO)</a>. Нажмите на окно
фильтрации по HPO, чтобы открыть форму поиска фенотипов. В форме в выпадающем списке представлены основные группы
фенотипов:

<p align="center">
<img src={require('/img/vv/hpo_search.png').default} width="800"/>
</p>

**Способы поиска и выбора термина для фильтрации:**
- Поиск термина непосредственно в выпадающем списке. С группами терминов можно
взаимодействовать с помощью кнопок <img src={require('/img/vv/show.png').default} height="15"/> и <img src={require('/img/vv/hide.png').default} height="10"/> для
раскрытия и сворачивания списка фенотипов группы, соответственно:

<p align="center">
<img src={require('/img/vv/list_of_phenotypes.png').default} width="270"/>
</p>

Кроме того, фенотип можно найти через поиск по названию или идентификатору
из HPO. Например, при поиске заболевания “glioma” находятся несколько терминов:

<p align="center">
<img src={require('/img/vv/glioma_search.png').default} width="800"/>
</p>

Для выбора термина для фильтрации нажмите на его строку. Можно выбрать как целую группу, так и
конкретный фенотип. Выбрав один из терминов (например, "Glioma"), вы увидите карточку этого заболевания:

<p align="center">
<img src={require('/img/vv/glioma_page.png').default} width="800"/>
</p>

Над карточкой заболевания отмечены фенотипы, выбранные для фильтрации. В скобочках указано количество
связанных с фенотипом генов. Нажав на термин, вы увидите карточку этого фенотипа. Термин можно удалить,
нажав на крестик <img src={require('/img/vv/delete_term_btn.png').default} width="15"/>:

<p align="center">
<img src={require('/img/vv/delete_term.png').default} width="500"/>
</p>

Слева в карточке фенотипа даны его определение и комментарий. В середине представлена иерархия заболевания:
выше выбранного заболевания Glioma находятся его родительские фенотипы
(те группы фенотипов, которые включают Glioma), а ниже - его дочерние
фенотипы (заболевания, которые входят в группу Glioma).
Справа представлены гены, ассоциированные с данным заболеванием (в данном случае было найдено 45 генов).

- Поиск термина по иерархии в карточке фенотипа: нажав на название родительского или дочернего
фенотипа, вы попадете на страницу этого фенотипа. Если фенотип ещё не выбран для фильтрации и
найдены ассоциированные с ним гены, то его можно выбрать для фильтрации вариантов,
нажав на кнопку <img src={require('/img/vv/add_term.png').default} width="150"/>.

- Поиск термина в режиме мультиселекта: для этого переключите режим поиска,
нажав на <img src={require('/img/vv/switch_mode.png').default} width="120"/> в левом верхнем углу окна.
Вы увидите список основных групп фенотипов с флажками для выбора.
Фенотип можно найти через поиск по названию или идентификатору
из HPO. Например, при поиске заболевания “glioma” находятся несколько терминов:

<p align="center">
<img src={require('/img/vv/multiselect_hpo.png').default} width="800"/>
</p>

В режиме мультиселекта можно выбрать сразу несколько терминов в выпадающем списке.
Для выбора термина для фильтрации нажмите на его строку:

<p align="center">
<img src={require('/img/vv/glioma_checked.png').default} width="400"/>
</p>

Чтобы закончить добавление терминов для фильтрации, кликните вне области выпадающего списка в
пределах формы поиска фенотипа. Наверху формы вы увидите фенотипы, выбранные для фильтрации. В скобочках
указано количество связанных с фенотипом генов. Нажав на термин, вы увидите карточку этого фенотипа.
Термин можно удалить, нажав на крестик <img src={require('/img/vv/delete_term_btn.png').default} width="15"/>.

<p align="center">
<img src={require('/img/vv/multiselect_mode.png').default} width="800"/>
</p>

Чтобы ***отфильтровать варианты*** по генам, связанным с выбранными фенотипами,
нажмите <img src={require('/img/custom-annot/ok_btn.png').default} width="130"/>.
Выбранные фенотипы можно увидеть во всплывающей подсказке, наведя курсор на поле фильтрации по HPO:

<p align="center">
<img src={require('/img/vv/chosen_phenotypes.png').default} width="300"/>
</p>

Для того чтобы ***отменить фильтрацию*** по всем фенотипам, наведите курсор на окно фильтрации по HPO и
нажмите на появившийся крестик:

<p align="center">
<img src={require('/img/vv/cancel_hpo.png').default} width="150"/>
</p>

или нажмите на поле и отредактируйте свой выбор в форме поиска фенотипов.

### 5. Шкала фильтрации по глубине секвенирования {#depth-scale}

- Фильтрация по минимальному
уровеню глубины. Можно варьировать уровень как с помощью плюса-минуса, так и с помощью слайдера.
Между 1x и 50x шаг фильтрации (цена деления) - 1, между 50x и 100x - 3, а между 100x и 300x - 7.

<p align="center">
<img src={require('/img/vv/seq_depth.png').default} width="350"/>
</p>

## Сброс фильтрации {#reset-filtering}

Для сброса ***всех*** примененных фильтров
нажмите на <img src={require('/img/vv/clear_filters.png').default} width="43"/>.

Для сброса ***отдельного фильтра***:
- Нажмите на окно фильтрации и затем на <img src={require('/img/vv/clear_option.png').default} width="100"/>.
- Если фильтр позволяет выбрать сразу несколько опций (например, фильтр по гену), то можно убрать одну из
выбранных опций фильтрации. Для этого нажмите на окно фильтрации и снимите галочку у опции.
- Для изменения выбора фенотипов HPO, по которым идёт фильтрация вариантов,
нажмите на поле фильтрации по HPO и отредактируйте свой выбор в форме
поиска фенотипов. Для сброса фильтрации по всем выбранным фенотипам наведите курсор на окно фильтрации по HPO и
нажмите на появившийся крестик:

<p align="center">
<img src={require('/img/vv/cancel_hpo.png').default} width="150"/>
</p>
