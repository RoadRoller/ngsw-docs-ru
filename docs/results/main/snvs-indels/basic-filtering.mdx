Базовые параметры фильтрации находятся сверху таблицы:

<p align="center">
<img src={require('/img/vv/basic_filtering.png').default} width="900"/>

:::info На заметку

### Основные параметры фильтрации {#basic-filtering-columns}

:::caution На заметку
На [странице "Настройки пользователя"](/settings/profile#basic-mode-filters)
можно поменять выбор основных параметров фильтрации. Здесь подробно рассказано о тех параметрах,
которые добавлены в настройках по умолчанию.
:::

1. <font size="4"><b>Тип мутации</b></font>: Somatic (соматическая), Germline (герминальная).

:::note На заметку
Параметр фильтрации "Тип мутации" не отображается в базовом режиме поиска в
[Variant Viewer с герминальными мутациями](/results/main/snvs-indels/variant-viewer).
:::

Как здесь, так и во всех последующих случаях, <b><i>алгоритм фильтрации</i></b> один:
- Нажмите на окно фильтрации;
- Из появившегося списка выберите нужную опцию фильтрации, либо введите интересующую вас опцию в окно
поиска и выберите из найденных.

Готово! Фильтр применен. То, какой выбран фильтр, можно увидеть в окошке (в данном случае выбран тип мутации
"Somatic"):

<p align="center">
<img src={require('/img/vv/set_option.png').default} width="150"/>
</p>

2. <font size="4"><b>Панель генов</b></font>. Свою панель генов можно добавить на странице "<b>Панели генов</b>",
о чем подробно написано в <a href="/settings/gene-panels">соответствующем разделе</a>,
либо можно импортировать панели из библиотеки самых распространенных клинических панелей. Для импорта
нажмите на "<b>Импорт из библиотеки</b>":

<p align="center">
<img src={require('/img/vv/gene_panel.png').default} width="300"/>
</p>

В открывшейся форме выберите нужные панели, отметив их галочками:

<p align="center">
<img src={require('/img/gene_panels/checked_panels.png').default} width="600"/>
</p>

и нажмите на <img src={require('/img/gene_panels/import_btn.png').default} width="170"/>, чтобы добавить панель.

3. <font size="4"><b>Ген</b></font>: фильтрация по транскрипту гена (приведен в скобках). Можно отфильтровать
варианты сразу по нескольким генам. Для этого отметьте галочками нужные гены:

<p align="center">
<img src={require('/img/vv/filter_by_gene.png').default} width="200"/>
</p>

:::note На заметку
Если для гена известно несколько транскриптов, в списке представлены все:
<p align="center">
<img src={require('/img/vv/many_transcripts.png').default} height="400"/>
</p>
:::

4. <font size="4"><b>HPO</b></font> - фильтрация по термину из словаря фенотипических аномалий
человека <a href="https://www.ebi.ac.uk/ols/ontologies/hp">Human Phenotype Ontology (HPO)</a>. Нажмите на окно
фильтрации по HPO, чтобы открыть форму поиска фенотипа. В форме в выпадающем списке представлены основные группы
фенотипов:

<p align="center">
<img src={require('/img/vv/hpo_search.png').default} width="800"/>
</p>


Можно отфильтровать варианты как по целой группе, так и по конкретному фенотипу.
Фенотип можно найти в списке терминов определенной группы фенотипов или через поиск по названию или идентификатору
из HPO. Например, при поиске заболевания “glioma” находится несколько терминов:

<p align="center">
<img src={require('/img/vv/glioma_search.png').default} width="800"/>
</p>

Выбрав одно из них (например, "Glioma"), вы окажетесь на странице этого заболевания:

<p align="center">
<img src={require('/img/vv/glioma_page.png').default} width="800"/>
</p>

Слева даны определение заболевания и комментарий. В середине представлена иерархия заболевания:
выше выбранного заболевания Glioma находятся его родительские фенотипы
(те группы фенотипов, которые включают Glioma), а ниже - его дочерние
фенотипы (заболевания, которые входят в группу Glioma).
Справа представлены гены, ассоциированные с данным заболеванием (в данном случае было найдено 45 генов).
Чтобы отфильтровать варианты по этим генам,
нажмите <img src={require('/img/gene_panels/ok_btn.png').default} width="130"/>.

Для того чтобы ***отменить фильтрацию***, нажмите на крестик в поле фильтрации по HPO:

<p align="center">
<img src={require('/img/vv/cancel_hpo.png').default} width="150"/>
</p>

5. <font size="4"><b>Шкала фильтрации по глубине секвенирования</b></font> - фильтрация по минимальному
уровеню глубины. Можно варьировать уровень как с помощью плюса-минуса, так и с помощью слайдера.

<p align="center">
<img src={require('/img/vv/seq_depth.png').default} width="350"/>
</p>

6. <font size="4"><b>Ваши запросы</b></font> - фильтрация по сохраненным запросам. То, как сохранить запрос,
описано в <a href="/results/main/snvs-indels/advanced-filtering#save-query">соответствующем разделе</a>.

### Сброс фильтрации {#reset-filtering}

Для сброса ***всех*** примененных фильтров
нажмите на <img src={require('/img/vv/clear_filters.png').default} width="43"/>.

Для сброса ***отдельного фильтра***:
- Нажмите на окно фильтрации и затем на <img src={require('/img/vv/clear_option.png').default} width="100"/>.
- Если фильтр позволяет выбрать сразу несколько опций (например, фильтр по гену), то можно убрать одну из
выбранных опций фильтрации. Для этого нажмите на окно фильтрации и затем на строку выбранной опции, так чтобы
у неё снялась галочка.
- Для сброса фильтра по HPO нажмите на крестик в поле фильтрации по HPO:

<p align="center">
<img src={require('/img/vv/cancel_hpo.png').default} width="150"/>
</p>
