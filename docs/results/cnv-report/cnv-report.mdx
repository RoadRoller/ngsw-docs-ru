# Отчёт по CNV

Отчёт по CNV - это отчёт по результатам анализа числа копий хромосом в исследованном образце или образцах пациента.
Если соблюдены все условия построения отчёта (см. ниже), в результате анализа формируется два
отчёта - **[отчёт для образца](/results/cnv-report/sample-cnv-report)**
и **[отчёт для пациента](/results/cnv-report/patient-cnv-report)**.

### Условия построения отчёта по CNV {#cnv-report-conditions}

Отчёт по CNV строится для образца, если выполняются следующие условия:
1. Образец загружен как ***образец неопухолевой ткани*** (образец, имеющий тип "NORMAL") в формате FASTQ или BAM.
2. ***Анализ образца успешно завершился*** (то есть все стадии анализа имеют статус "Завершено"). Если анализ
завершился с ошибкой, то в "Отчёте по CNV" для пациента будет соответствующее сообщение для образца, а "Отчёт
по CNV" для образца не построится.
3. Для образца ***успешно завершилась стадия "Выявление вариации числа копий"***. Если стадия завершилась
с ошибкой, то в "Отчёте по CNV" для пациента будет соответствующее сообщение для образца, а "Отчёт по CNV" для образца
не построится. Если стадия исключена из анализа, то отчёт для образца не построится, а в отчёте для пациента образец
упомянут не будет. Если стадия выключена у всех образцов пациента, то "Отчёт по CNV" для пациента не построится.
3. Шаблон отчёта, включающий блоки "Герминальные CNV: результаты" и/или "Герминальные CNV: интерпретация",
активен (регулируется на странице "[Шаблоны отчётов](/settings/report-templates/report-templates)").
4. Шаблон отчёта был добавлен в систему до того, как был обработан образец.

:::info На заметку
Если у вас нет добавленного шаблона "Отчёт по CNV" и он вам необходим для анализа,
вы можете обратиться к администратору или добавить его самостоятельно,
как описано [здесь](/settings/report-templates/main-templates#add-cnv-report).
:::
