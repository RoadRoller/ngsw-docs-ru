# Отчет по CNV

Отчет по CNV - это отчет по результатам анализа числа копий хромосом в исследованном образце или образцах пациента.
Отчет строится только ***для образцов здоровой ткани*** (образцов, имеющих тип "NORMAL").
В результате анализа формируется два 
отчета - **[отчет для образца](/results/cnv-report/sample-cnv-report)** 
и **[отчет для пациента](/results/cnv-report/patient-cnv-report)**.
