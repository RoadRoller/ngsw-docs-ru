# Отчет по CNV

Отчет по CNV - это отчет по результатам анализа числа копий хромосом в исследованном образце или образцах пациента.
Если соблюдены все условия построения отчета (см. ниже), в результате анализа формируется два
отчета - **[отчет для образца](/results/cnv-report/sample-cnv-report)** 
и **[отчет для пациента](/results/cnv-report/patient-cnv-report)**.

### Условия построения отчета по CNV

Отчет по CNV строится для образца, если выполняются следующие условия:
1. Образец загружен как ***образец неопухолевой ткани*** (образцов, имеющих тип "NORMAL") в формате FASTQ или BAM.
2. ***Анализ образца успешно завершился*** (то есть все стадии анализа имеют статус "Завершено"). Если анализ
завершился с ошибкой, то в "Отчете по CNV" для пациента будет соответствующее сообщение для образца, а "Отчет
по CNV" для образца не построится.
3. Для образца ***успешно завершилась стадия "Выявление вариации числа копий"***. Если стадия завершилась
с ошибкой, то в "Отчете по CNV" для пациента будет соответствующее сообщение для образца, а "Отчет по CNV" для образца
не построится. Если стадия исключена из анализа, то отчет для образца не построится, а в отчете для пациента образец
упомянут не будет. Если стадия выключена у всех образцов пациента, то "Отчет по CNV" для пациента не построится.
3. Шаблон отчета активен (регулируется на странице "[Шаблоны отчетов](/settings/report-templates/report-templates)").
4. Шаблон отчета был добавлен раньше, чем был обработан образец.

:::info На заметку
Если у вас нет добавленного шаблона "Отчет по CNV" и он вам необходим для анализа,
вы можете обратиться к администратору или добавить его самостоятельно,
как описано [здесь](/settings/report-templates/main_templates#добавление-шаблона-отчета-по-cnv).
:::
