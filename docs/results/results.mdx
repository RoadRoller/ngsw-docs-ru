# Страница образца

На странице образца приведена информация о ходе анализа образца, тут можно изменить параметры анализа и посмотреть его
результаты после завершения обработки.
Страницу образца можно открыть:
- со страницы "Образцы", нажав на имя образца:

<p align="center">
<img src={require('/img/rus/sample_row_on_samples_page.png').default} width="900"/>
</p>

- со [страницы пациента](/add-patient#patient-page), нажав на строку образца:

<p align="center">
<img src={require('/img/rus/tumor_sample_row.png').default} width="900"/>
</p>

- со [страницы запуска](/runs#run-page), нажав на строку образца:

<p align="center">
<img src={require('/img/rus/sample_of_run.png').default} width="900"/>
</p>

Вы попадете на [вкладку **"Главное"**](/results/main/main) образца, где расположены информация об образце,
основные результаты и стадии процесса анализа.

<p align="center">
<img src={require('/img/rus/tumor_main_page.png').default} width="900"/>
</p>

Ход анализа (и, соответственно, наличие результатов) зависит от выбранных настроек анализа
и типа образца (образец опухолевой или неопухолевой ткани).

Со вкладки "Главное" можно перейти на остальные вкладки на странице образца:

1. **Вкладки с построенными отчётами** - наличие вкладок зависит от наличия активных шаблонов отчётов (регулируется
на [странице "Шаблоны отчётов"](/settings/report-templates/report-templates) в блоке страниц "Настройки"), их
применимости к данному образцу и наличия необходимых для построения отчёта результатов. По умолчанию для вас могут
быть активными [следующие отчёты](/settings/report-templates/main-templates) (добавление этих отчётов можно
запросить у администратора системы):

- **[Отчёт по CNV](/results/cnv-report/cnv-report)** - отчёт по результатам анализа числа копий хромосом в образце.
- **[Отчёт для анализа глиомы](/results/glioma-report)** - отчёт с соматическими и герминальными вариантами,
связанными с глиомами.

2. **[Биоинформатический отчёт](/results/bioinformatic-report)** - включает статистику о файле и отчёты
о выполненных стадиях анализа образца (отчёт о качестве, отчёт по выравниванию, отчёт по покрытию и отчёт по
выявленным SNVs/Indels).

3. **[Анализ покрытия](/results/coverage-analysis)** (вкладка видна только для образцов, загруженных в форматах
FASTQ или BAM, и только после успешного завершения всех стадий анализа) - содержит информацию о геномном покрытии
и покрытии генов и транскриптов в образце.

4. **[Детали процесса анализа](/results/workflow-details/workflow-details)** - включает подробную информацию о
процессе обработки образца и промежуточные данные.

5. **[Параметры](/results/parameters)** (вкладка видна только для образцов, загруженных в форматах FASTQ или BAM) -
содержит информацию о параметрах анализа образца.
