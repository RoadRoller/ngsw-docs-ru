# Результаты анализа

После завершения анализа образца его результаты расположены на странице образца.
Чтобы их посмотреть, откройте страницу образца, кликнув на соответствующую строку на странице пациента (или запуска):

<p align="center">
<img src={require('/img/rus/tumor_sample_row.png').default} width="900"/>
</p>

Вы попадете на [вкладку "**Главное**"](/results/main/main) образца, где расположены информация об образце, основные результаты
и стадии процесса анализа.

<p align="center">
<img src={require('/img/rus/tumor_main_page.png').default} width="900"/>
</p>

Ход анализа (и, соответственно, наличие результатов) зависит от выбранных настроек анализа
и типа образца (образец опухолевой или неопухолевой ткани).

Со вкладки "Главное" можно перейти на остальные вкладки на странице образца:
1. **Вкладки с построенными отчетами** - наличие вкладок зависит от наличия активных шаблонов отчетов (регулируется
на странице "Шаблоны отчетов" в блоке страниц "Настройки"), их применимости к данному образцу и наличия
необходимых для построения отчета результатов. По умолчанию для Вас могут быть активными следующие отчеты (добавление
этих отчетов можно запросить у администратора):
- **[Отчет по CNV](/results/cnv-report/cnv-report)** - отчет по результатам анализа числа копий хромосом в образце.
- **Отчет для анализа глиомы** - отчет с соматическими и герминальными вариантами, связанными с глиомами.
2. **[Биоинформатический отчет](/results/bioinformatic-report)**
3. **[Анализ покрытия](/results/coverage-analysis)**
4. **[Детали процесса анализа](/workflow-details/workflow-details)**
5. **Параметры**
