<span> Тип секвенирования, как правило, определяется Genomenal автоматически.
Однако, если Вы уверены в том, какой тип секвенирования был использован при получении Ваших данных,
выберите соответствующий тип для ускорения анализа. Это может быть полногеномное секвенирование (WGS) или
таргетное секвенирование (экзом или панель).
Обратите внимание, что, определея тип секвенирования, Вы выбираете его сразу для всех образцов {props.set}.</span>
